# Product Requirement Document (PRD) - mycost（进阶分析记账）

## 1. 产品定位
一款偏“进阶分析”的个人财务管理应用。深度集成 iOS 原生特性，专注高效记账与清晰的收支洞察。

## 2. 核心功能
### 2.1 账目管理
- 快速记账：支持支出/收入切换、金额输入、分类选择、日期调整与备注。
- 金额输入：禁用系统键盘，使用内置“计算器式输入面板”，支持直接计算后录入。
- 保存入口：移动到计算器面板右下角，按钮文案为“完成”。
- 布局要求：金额显示位于计算器面板上方；分类以图标网格展示，末尾提供“添加”入口跳转到分类管理。
- 账目列表：卡片式布局，按日期分组展示账目。
- 操作交互：左滑删除，点击卡片进入编辑。

### 2.2 分类系统
- 预设分类：
  - 支出：餐饮、交通、购物、娱乐、医疗、居家、教育等。
  - 收入：工资、奖金、理财收入、其他。
- 图标方案：统一使用 Apple SF Symbols。
- 分类管理：支持新增、删除、修改分类；可修改分类名称与图标。

### 2.3 数据统计
- 顶部切换：支持左右滑动切换「月/年」周期，并显示当前周期时间（如：2026年1月）。
- 收支分析：统计页顶部新增「收支分析」卡片，展示收入/支出双柱月度趋势（交互与样式详见 UI 规范）。
  - 年统计页顶部同样展示收支趋势，按年汇总展示。
  - 年趋势默认可视 6 年，可向左滑动查看更早年份数据。
  - 点击趋势柱体后，下方 Donut 与分类明细切换到对应月份/年份数据。
- 构成分布：Donut Chart（中心镂空），外侧标注分类图标与百分比。
- 构成分布支持「支出/收入」切换，分类列表随之联动更新。
- 详细清单：卡片式分类列表，展示分类图标、名称、交易笔数与总金额。
- 空状态：当选定月份无账目时，中下部显示“本月暂无收支记录”占位提示。

### 2.4 备份与同步
- iCloud 同步：基于 CloudKit 的多设备自动同步。

## 3. 视觉与体验 (UI/UX)
- 风格参考：iCost。大圆角卡片、清晰留白。
- 深色模式：全面适配系统深色/浅色模式。
- 主题设置：支持「跟随系统 / 浅色 / 深色」三档切换。
- 触感反馈：保存、删除等关键操作加入轻触反馈。
